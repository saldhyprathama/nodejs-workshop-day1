🎬 Video Streaming

Streaming samples/videoplayback.mp4 in 5 MB chunks using HTTP Range requests and fs.createReadStream.

EndpointGET /api/videos/stream
Chunk size5 MB = 5 × 1024 × 1024 bytes
Methodfs.createReadStream(VIDEO_PATH, { start, end }).pipe(res)
ProtocolHTTP 206 Partial Content + Accept-Ranges: bytes
